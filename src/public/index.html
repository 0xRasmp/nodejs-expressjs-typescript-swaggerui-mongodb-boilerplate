Node.js Express TypeScript MongoDB Boilerplate

A production-ready RESTful API boilerplate built with Node.js, Express, TypeScript, and MongoDB

Quick Start

# Clone the repository
git clone https://github.com/yourusername/nodejs-express-typescript-mongodb-boilerplate.git

# Install dependencies
yarn install

# Set up environment variables
cp .env.example .env

# Start development server
yarn dev

Features

TypeScript

Built with TypeScript for better development experience and type safety

Express.js

Fast, unopinionated, minimalist web framework for Node.js

MongoDB

MongoDB integration with Mongoose ODM for data modeling

Swagger UI

Interactive API documentation with Swagger UI

Environment Variables

Environment variables support with dotenv

Hot Reload

Development server with hot reload using nodemon

Available Scripts

yarn dev # Start development server
yarn build # Build for production
yarn start # Start production server

Project Structure

src/
├── controllers/ # Request handlers
├── models/ # MongoDB models
├── routes/ # API routes
├── types/ # TypeScript type definitions
├── public/ # Static files
├── index.ts # Application entry point
