Node.js Express TypeScript MongoDB Boilerplate

A production-ready RESTful API boilerplate built with Node.js, Express, TypeScript, and MongoDB

Quick Start

# Clone the repository
git clone https://github.com/yourusername/nodejs-express-typescript-mongodb-boilerplate.git

# Install dependencies
yarn install

# Set up environment variables
cp .env.example .env

# Start development server
yarn dev

Authentication API

Available Endpoints:

POST /api/auth/login - Login with access token (no password)
POST /api/auth/me - Get current user info

Simple authentication system - just provide your access token to login. No passwords required!

Token API

Available Endpoints:

POST /api/tokens - Generate new access token
GET /api/tokens/validate/:token - Validate token & get user info
GET /api/tokens - Get all tokens (paginated)
PATCH /api/tokens/:id/deactivate - Deactivate token

Generate secure access tokens like 966f7dbf481761363f9b94fae. The token itself serves as the user ID - anyone with the token can identify themselves. Set expiration in days (e.g., 30 for 30 days).

Twitter Username API

Available Endpoints:

POST /api/twitter-usernames - Add Twitter username to access token
GET /api/twitter-usernames/token/:accessToken - Get usernames for a token
GET /api/twitter-usernames/username/:username - Get tokens for a username
DELETE /api/twitter-usernames/remove - Remove username from token

Each access token can only add a specific Twitter username once. Multiple tokens can add the same username, but one token cannot add the same username twice.
